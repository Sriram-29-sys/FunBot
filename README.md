# FunBot
📌 Project Overview:

FunBot is a conversational AI chatbot built with Python that processes user input and generates dynamic, human-like responses in real time.
The project demonstrates basic Natural Language Processing concepts and real-time user interaction.
🎯 Objective:

Build a chatbot from scratch using Python
Understand conversation flow handling
Implement response generation logic
Practice clean coding and project structuring
⚙️ Features:

Interactive command-line conversation
Rule-based response generation
Easy-to-modify response patterns
Lightweight and beginner-friendly
Structured and readable code
🛠 Tech Stack

Python
Basic NLP concepts
Standard Python libraries
